01-Controller, Domain Objects

  • ng-app - to bind the application module to the html page
  • ng-controller - to bind the controller to the div
  • ng-module - to bind the control value to the controller property => two way binding

{{vm.title}}

App.ts = ng-app

index.html file has body with ng-app="SimpleModule" It defines we will use main module( = application) with name "SimpleModule". Modules in typescript are the same like namespace in c#. It defines the scope of the module. We van use import/export to use class or interface from different namespace. 

module App { 
    angular.module("SimpleModule", []);
}

SimpleController.ts = ng-controller

div on the index.html has ng-controller="SimpleController" This Specifies we would like to use the "SimpleController" controller of the given "SimpleModule" module 

module App.Simple {
    class SimpleController {
        title: string;
        constructor() {
            this.title = "Simple Controller";
        }
    }
    angular.module("SimpleModule", [])
        .controller("SimpleController", SimpleController);
}

Input = ng-module

input tag has defined two way binding with "SimpleController" by using ng-module inside the input tag 

                    
                    input id="controllerTitle" type="text" ng-model="vm.title"
